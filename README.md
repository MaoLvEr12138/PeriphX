# PeriphX

**Configurable FPGA peripherals for MCU developers.**

PeriphX is an open-source project that aims to make FPGA-based peripherals accessible to MCU developers.

The long-term vision is to allow developers to describe the peripherals and accelerators they need through simple configuration files, then automatically generate FPGA bitstreams and corresponding MCU SDKs. The goal is to make FPGA resources feel like configurable hardware extensions rather than a separate development domain.

Potential future components include:

- UART
- SPI
- I2C
- PWM
- GPIO
- Timers
- DSP accelerators
- Custom hardware modules

## Current Status

⚠️ PeriphX is in a very early stage of development.

The architecture, configuration format, component model, and build system are still under active design. Nothing should be considered stable yet.
